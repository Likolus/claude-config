# Professional Consulting Analysis Skill (from DeerFlow)

**Ported from DeerFlow to Claude Code**

This skill produces professional, consulting-grade research reports covering domains such as **market analysis, consumer insights, brand strategy, financial analysis, industry research, competitive intelligence, and investment research**.

## When to Use This Skill

**Always use this skill when:**

- User requests a market analysis, consumer insight report, or industry research
- User needs a professional consulting-style research report
- User provides a research subject and needs a structured analysis framework
- The task involves transforming research findings into structured strategic narratives

## Two-Phase Workflow

### Phase 1: Analysis Framework Generation

Given a research subject, produce a complete **analysis framework** that serves as the blueprint for data collection and final report generation.

**Framework Components:**

1. **Chapter Skeleton** - Hierarchical structure with clear scope
2. **Data Requirements** - Specific data points needed per chapter
3. **Analysis Logic** - Analytical frameworks to apply
4. **Visualization Plan** - Chart types for each data point

### Phase 2: Report Generation

After data collection, synthesize findings into a final polished report following McKinsey/BCG consulting standards.

## Analysis Flow Pattern

For each subsection, follow: **Visual Anchor → Data Contrast → Integrated Analysis**

Use the **"Data → User Psychology → Strategy Implication"** chain for insights.

## Consulting Voice Standards

- **Precise**: Use specific numbers ("increased 23%" not "increased significantly")
- **Structured**: Clear hierarchy, numbered lists, logical flow
- **Evidence-based**: Every claim backed by data or source
- **Action-oriented**: Insights lead to recommendations
- **Executive-friendly**: Clear executive summary, visual hierarchy

## Data Authenticity Protocol

**CRITICAL**: All data MUST be from provided sources. NO hallucinations.
