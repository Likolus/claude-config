name: Install tools autonomously
description: User allows autonomous installation of required tools
type: feedback
---

Если для работы нужен инструмент, которого нет в системе, можно устанавливать его самостоятельно без запроса разрешения.

**Why:** Пользователь хочет, чтобы работа шла быстрее и эффективнее, без остановок на запросы установки инструментов.

**How to apply:** 
- Когда обнаруживается отсутствие необходимого инструмента (gh, curl, jq и т.д.), устанавливать его через winget или другой менеджер пакетов
- Информировать пользователя о том, что инструмент устанавливается
- Не запрашивать подтверждение на установку стандартных dev-инструментов
